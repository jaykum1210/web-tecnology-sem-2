CSS stands for cascading style sheet; this is use for formatting the web page. We have three types of stylesheet.
  1. Internal StyleSheet
  2. External StyleSheet
  3. Inline StyleSheet

Internal StyleSheet

Internal stylesheet embeded with html in the same html page

External StyleSheet

We create a seperate .css file for stylesheet and link this file with html code file. A single .css file can link with many HTML files.

Inline StyleSheet

This stylesheet works for a particular tag.
